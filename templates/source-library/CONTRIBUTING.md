# Contributing to This Source Library

Only submit entries that the source library is allowed to publish.

Every `downloadable` entry must include:

- a clear title and platform;
- a valid `http` or `magnet` source;
- expected file extensions;
- checksums where the Fusion Launcher schema requires them;
- evidence that the content can be redistributed, such as an author page,
  license, release page, or explicit permission.

Use `user_provided` for commercial ROMs, disc images, BIOS files, firmware,
keys, and any other content that users must provide themselves. Do not submit
download links for those entries.

Use `metadata_only` when you want compatibility notes or catalog metadata
without publishing a payload source.

Maintainers may reject entries that are unverifiable, incomplete, misleading,
or unsafe for users to connect.
